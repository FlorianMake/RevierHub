5 — phone (Fairphone / any android)
13
Enable developer mode + USB debugging manual
Settings → About Phone → tap Build Number 7×
Settings → System → Developer Options → USB Debugging ON
14
udev rule for ADB
echo 'SUBSYSTEM=="usb", ATTR{idVendor}=="18d1", MODE="0666", GROUP="plugdev"' | sudo tee /etc/udev/rules.d/51-android.rules sudo udevadm control --reload-rules && sudo udevadm trigger
idVendor 18d1 = Google/Android. Check yours with lsusb if different phone.
15
Verify
adb devices
Phone must show "Allow USB debugging?" popup → tap Allow
6 — known gotchas
⚠
Always use Qt 6.7.3 kit, disable 6.9 for this project
Projects → disable all kits except Qt 6.7.3 for Android arm64-v8a
⚠
lib_name manifest issue → always do clean rebuild
Cached builds can strip android.app.lib_name — Build → Clean All before deploy
⚠
MapLibre plugin name is "maplibre" not "maplibregl"
QML: Plugin { name: "maplibre" } — confirmed via QGeoServiceProvider::availableServiceProviders()
⚠
Map tile style: openfreemap works without API key
https://tiles.openfreemap.org/styles/liberty
