################################
#   Scrolling on the map.
################################
- while using the app the user shall be able to focus on walking / hunting or anything else,
so usually the screen follows the user position within a specified range. Repositioning will
happen every other 10 - 20 meters, this needs to be decided or maybe it will change after
the zoom lvl of the user.

- if the user want to scroll around, the repositioning will be turned off for some time, the
map center will stay on the position the user decided. If the user is stationary for a longer
time, the recentering mode wont turn on by itself. But the user will see a little marker on
the edge of the map where he is located, if he is out of the current view.

-If the user starts walking the recentering will turn on again. (maybe some more distance is
needed than we us to determine if a ne trail point is created)

be increased accordingly to the user interaction.
